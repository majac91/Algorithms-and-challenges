Frequency counter pattern

Write a function called 'same', which accepts two arrays. The f should return true if every value in the array has it's corresponding value squared in the second array. The frequency of values must be the same, but the order doesn't matter

Examples

  • same([1,2,3], [4,1,9]) //true
  • same([1,2,3], [4,1,]) //false
  • same([1,2,1], [4,4,1]) //false (must be the same frequency)
